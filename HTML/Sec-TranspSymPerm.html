Section 2.1 Sept. 29: Transposes, Symmetry, and Permutations

Subsection 2.1.1 Warmup

As usual, we should set up our working environment. There should be no output.

Warmup 2.1.1.
Use Sage to calculate an \(LDU\) factorization for the given matrix \(A\text{.}\) Recall the \(A=LU\) factorization is found with the command P, L, U = A.LU(pivot='nonzero').

Subsection 2.1.2 Exploring Symmetry

Exploration 2.1.1.
Create two symmetric matrices (probably of different sizes, just to be safe). Use the .inverse() command to find their inverses. What do you notice? What do you wonder?
Exploration 2.1.2.
Create a matrix \(A\text{.}\) Use the .transpose() command and multiply \(A A^T\) and \(A^T A\text{.}\) What do you notice? What do you wonder?

An advantage of working with symmetric matrices is in the elimination process. We really only need to work with half of the matrix (plus the main diagonal)! So let's see how symmetric matrices carry through to the \(LDU\) factorization.

Exploration 2.1.3.
Consider the matrix \(B\) below.
  1. Confirm that \(B\) is symmetric by using the .is_symmetric() command.
  2. Capture the matrices \(P\text{,}\) \(L\text{,}\) and \(U\text{,}\) using P, L, U = B.LU(pivot='nonzero'). Confirm that no row swaps were needed.
  3. Find the diagonal matrix \(D\) and a new matrix \(U_{\text{new}}\) that satisfies \(B = LDU_{\text{new}}\text{.}\)
  4. What do you notice? Choose a different symmetric matrix and run the calculations again to see if the same behavior persists.

Subsection 2.1.2.1 Fun with Permutations

Exploration 2.1.4.
Strang writes \(PA = LU\text{,}\) but Sage calculates the \(P\) that satisfies \(A = PLU\text{.}\) Let's explore their relationship (though maybe you know what it is already?).
  1. Given the \(A\) below, use P, L, U = A.LU() to calculate an \(LU\) decomposition.
  2. Confirm that \(A = PLU\text{.}\)
  3. What permutation matrix \(P'\) satisfies \(P' A = LU\text{?}\) Confirm it in Sage.

To close the section, let's put everything together in an exploration of the deep structure of a system of equations.

Exploration 2.1.5.
Consider the following system of 3 linear equations in 3 unknowns.
\begin{equation*} \left\{ \begin{array}{rrrrrrr} -x & - & 2y & + & 2z & = & 1 \\ 2x & + & 4y & - & z & = & 5 \\ x & - & 2y & - & 3z & = & 3 \end{array}\right. \end{equation*}
  1. Find \(D\) and \(U\) so that you can write \(PA = LDU\) and \(A = P^T LDU\text{.}\)
  2. Multiply \(P b\) to get \(b'\text{.}\)
  3. Solve \(Lc = b'\text{.}\)
  4. Solve \(Dy=c\text{.}\)
  5. Now solve the upper-triangular system \(Ux = y\text{.}\)
  6. What is the solution to the original system?
  7. Confirm your solution by using Sage's .rref() command on an appropriate matrix.

Exercises 2.1.2.2 Exercises

1.
Find an example of a matrix \(A\) such that \(A^T A = 0\text{,}\) but \(A \neq 0\text{.}\)
2.
These are true or false questions. If the statement is true, explain why you know it is true. If the statement is false, give an example that shows it is false.
  1. The block matrix \(\left( \begin{smallmatrix} A & 0 \\ 0 & A \end{smallmatrix}\right)\) is automatically symmetric.
  2. If \(A\) and \(B\) are symmetric, then their product \(AB\) is symmetric.
  3. If \(A\) is not symmetric, then \(A^{-1}\) is not symmetric.
3.
For each of the following, find an example of a \(2\times 2\) symmetric matrix with the given property:
  1. \(A\) is not invertible.
  2. \(A\) is invertible but cannot be factored into \(LU\text{.}\)
  3. \(A\) can be factored into \(LDL^T\text{,}\) but not into \(LL^T\) because \(D\) has negative entries.
4.

This is a new factorization of \(A\) into triangular times symmetric:

Start with \(A = LDU\text{.}\) Then \(A = B S\text{,}\) where \(B = L\left(U^T\right)^{-1}\) and \(S = U^T D U\text{.}\)

Explain why this choice of \(B\) is lower triangular with \(1\)'s on the diagonal. Expain why \(S\) is symmetric.
